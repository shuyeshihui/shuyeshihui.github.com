最近的困惑

By Tilly

Feb 28 2017 Updated:May 7 2017

今天刚好是在yhouse做产品整整一年，晚上给几个团队部门的负责人发微信说感谢他们这一年来对我的支持和帮助，也希望他们能给我提一些建议。但是总体来说，大家大部分都在说我这一年进步很大，成长飞速这类激励的话，也说了一些建议，但是都是我已知的方面要做改进的。比如，提升协调各个部门高效工作的方法，在业务逻辑需求阐述方面希望更加全面和具体等。这些是我总结出来已知的问题，其实还有一些，比如更清晰的阐述自己的观点和见解，提升解决方案的深刻背景认知的程度，权衡目标和数据确定优先级上面等，都还有一定提高空间的。

不知道为什么最近有这样一个困扰，觉得产品经理越往后期做到底是做什么？我该怎么去发展？出去创业？管团队？做培训？做自媒体？看看目前我们熟知的一些互联网产品大牛的现状，无非就这么几种，一个非常有趣的现象，在大公司里面，针对技术岗位有完善的可衡量的专业晋升通道，也有管理晋升通道；但是产品岗，到了5-10年的位置，到了这个阶段，专业能力基本上都差不多了（再提升其实是人的视野等其它方面能力的提示了），大家都变成去带团队和项目了，其实只有管理岗位的晋升通道。产品经理可量化衡量的标准还是很难定义的，因为出色的产品靠的是天时地利人和，所以用出色的产品来衡量太难了。很多时候大家做的只是一款普通的产品，不温不火，甚至是款失败的产品，上线一年半载即淡出用户视野匆匆退出市场。

之前听过一个朋友说的案例，说他的一个产品朋友干了几年就不做了，回县城开实体店了，把产品用户思维新媒体运营用在实体店的经营上。心想，这未尝不是一种思路，而且在产品经理行业中的知识和思维带到了实体行业中。前几个月订阅的李笑来的专栏，说财富自由和认知升级，有个观点的大意是说我们没有必要专注于一个领域上，而是选择多个自己擅长的领域全面发展，就好比说你只擅长一个领域，你最多拿个95分吧；而我在这个领域只能拿个80分，但是我又擅长投资领域，可以拿60分，也懂一些编程，也可以拿个50分。80乘以60乘以50远远大于95。这种说法好比高中作文会探讨的议论文话题，专才重要还是全才重要。专才难，全才也难。为什么不选择几个擅长的领域，每个领域都做到及格，这也是大才能呀。我记得当时这个观点给我的影响就是，选择了编程和股票投资领域，开始了学习，这几个月竟也收获了不少知识带给我的价值回报。

我的困扰，是出于对于我从事这个行业（产品经理）可持续性以及未来发展前景的困扰，甚至想过是不是过个十年八年这个行业也就会消失了，毕竟小公司拍板的还是老板呀，产品经理充当的只不过是个项管和方案策划设计师的角色，那么在大公司最多也就是个方案策划设计师了。产品的专业能力上是可以封顶的，最近几年产品经理的系统培训层出不穷，大体都差不多，需求定位市场规划沟通协调运营落地数据目标导向迭代管理等等。那么在这接下去的一段时间里，在做好产品本职的同时，要去探索是不是可以产品经理+，扩展我其它领域的知识学习。先去尝试一段时间吧，知识肯定是越多越好呢。
